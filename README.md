# Fitness Tracker VK Bot

**Fitness Tracker** — это VK-бот для отслеживания физической активности с элементами геймификации. Бот позволяет пользователям записывать тренировки, следить за прогрессом, участвовать в рейтингах и открывать новые стили упражнений за очки опыта (XP).

Проект разработан в рамках лабораторных работ по дисциплине «Основы управления IT-проектами» (ВолгГТУ, группа ПрИн-367).

---

## Функциональность

### Учёт тренировок
- Поддержка 4 базовых упражнений: подтягивания, отжимания, приседания, пресс.
- Различные стили выполнения (например, «классические», «с весом», «алмазные»).
- Валидация вводимых данных и защита от ошибок ввода.

### Геймификация
- **Система XP:** начисление очков опыта за каждую выполненную тренировку.
- **Магазин стилей:** возможность покупать новые, более сложные стили упражнений за накопленные XP.
- **Разблокировка:** прогрессивное открытие контента по мере тренировок.

### Статистика и аналитика
- Личная статистика за день, неделю и месяц.
- Глобальный рейтинг пользователей по общему опыту.
- Рейтинги по конкретным упражнениям (топ-10 по максимальным повторениям).

### Удержание пользователей
- Настраиваемые ежедневные напоминания о тренировках.
- Мотивационные сообщения при побитии личных рекордов.
- Поддерживающие сообщения при пропуске тренировок.

## Установка и запуск

### 1. Требования
Убедитесь, что у вас установлен Python версии 3.11 или выше.

### 2. Клонирование репозитория
```bash
git clone https://github.com/your-username/fitness_tracker_bot.git
cd fitness_tracker_bot
```

### 3. Создание виртуального окружения

Windows:

```bash
python -m venv .venv
.venv\Scripts\activate
```
macOS / Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 4. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 5. Настройка окружения

Создайте файл .env в корневой папке и добавьте в него токен сообщества и ID группы:

```text
VK_TOKEN=vk1.a.YOUR_TOKEN_HERE
GROUP_ID=123456789
```
Примечание: Токен должен быть типа Community Token (начинается на vk1.a.) с правами на доступ к сообщениям сообщества.

### 6. Запуск бота

```bash
python main.py
```
После запуска бот будет ожидать входящие сообщения через Long Poll API.
